Configure scene composition exports with collection-based organization and multi-layout support.

Overview

The Layout Setup panel defines how your Blender scene is organized and exported as USD scene compositions. Layouts reference your mesh assets and define their placement, rotation, and scale in the world.

Layout Configuration

Layout List

BlendMHS supports multiple layouts from a single Blender file. Each layout corresponds to a separate USD scene composition file.

  • Add Layout: Create a new layout configuration
  • Remove Layout: Delete the selected layout
  • Rename: Double-click to rename a layout

Collection Assignment

Each layout is linked to a Blender collection. All objects within that collection (and its children) will be included in the layout export.

Best Practices

  • Use descriptive collection names (e.g., "Level_01_Layout")
  • Organize assets into sub-collections for clarity
  • Keep shared assets in a separate "Assets" collection

Multi-Layout Support

Multiple Scenes from One File

Create multiple layouts for different purposes:

  • Level Variants: Day/night versions of the same level
  • Development Layouts: Test scenes for specific features
  • Asset Showcases: Isolated views of asset groups
