üé¨Animation Setup

Skeletal animation export with UsdSkel support, action selection, and timeline integration.

Overview

The Animation Setup panel configures skeletal animation export using USD's UsdSkel schema. Export armatures with skin weights and multiple animation actions for use in Meta Horizon Studio.

Panel Location

3D Viewport ‚Üí Sidebar (N) ‚Üí BlendMHS ‚Üí Animation Setup

Armature Configuration

Armature Selection

Select the armature to export. The armature and all meshes skinned to it will be included in the animated USD file.

Requirements

  • Armature must have at least one bone
  • Skinned meshes must have valid vertex groups matching bone names
  • At least one action should be assigned or available

Bone Hierarchy

BlendMHS exports the complete bone hierarchy with proper parent-child relationships. The skeleton structure is preserved exactly as defined in Blender.

Action Selection

Selective Action Export

Choose which animation actions to include in the export. Each action becomes a separate animation clip in USD.

Action List

The panel displays all available actions with enable/disable toggles:

  • Enable All: Include all actions in export
  • Disable All: Clear all action selections
  • Individual Toggle: Enable/disable specific actions

Frame Range

Configure the frame range for each action:

  • Use Action Range: Export the full action keyframe range
  • Custom Range: Specify start and end frames manually
  • Scene Range: Use the scene's timeline range

Best Practices

  • Name actions descriptively (e.g., "Idle", "Walk", "Run")
  • Keep bone counts reasonable for VR performance
  • Test animations in MHS after export to verify playback
  • Use consistent frame rates between Blender and MHS
