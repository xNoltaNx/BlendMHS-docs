‚ö°Quick Export

One-click export tools for rapid iteration and hot-reload workflow integration with Meta Horizon Studio.

Overview

The Quick Export panel provides streamlined export buttons for fast iteration. Instead of configuring options each time, Quick Export uses your preset configurations to export with a single click.

Panel Location

3D Viewport ‚Üí Sidebar (N) ‚Üí BlendMHS ‚Üí Quick Export

Export Actions

Export All

Exports everything in one operation: layouts, meshes, materials, and textures. This is the recommended action for full scene updates.

Recommended

Export Layout Only

Exports only the scene composition (layout) file. Use this when you've only moved or added instances without modifying the underlying meshes.

Export Selected

Exports only the currently selected objects. Useful for updating individual assets without re-exporting the entire scene.

Hot-Reload Workflow

Rapid Iteration with MHS

When working with Meta Horizon Studio's hot-reload feature, Quick Export becomes incredibly powerful:

  • Make changes in Blender
  • Click Quick Export (or save with auto-export enabled)
  • MHS automatically detects the updated files
  • See your changes in the VR preview instantly

Export Cache

BlendMHS uses a smart caching system to skip unchanged files during export. This dramatically speeds up exports when only a few assets have changed.

Cache Statistics

After export, the panel displays statistics showing how many files were exported vs. skipped due to caching.
