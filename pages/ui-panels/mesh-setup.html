Overview

The Mesh Setup panel controls how individual mesh objects are exported to USD. Configure LOD generation, UV handling, vertex colors, and modifier application for each mesh.

Mesh Export Options

Apply Modifiers

When enabled, all modifiers on the mesh will be applied during export, baking the result into the USD file.

  • Enabled: Modifiers are applied (subdivision, booleans, etc.)
  • Disabled: Base mesh is exported without modifier effects

Export UVs

Controls UV coordinate export. Multiple UV sets are supported.

UV Set Naming

UV sets are exported with their Blender names preserved for material mapping in MHS.

Export Vertex Colors

When enabled, vertex color data is exported as displayColor primvars in USD.

Auto-Detection

BlendMHS automatically detects if a mesh has vertex colors and includes them in the export.

LOD Configuration

Level of Detail Support

BlendMHS supports both manual and auto-generated LODs with USD variant sets.

Manual LODs

Create LOD meshes manually in Blender using the naming convention:

MeshName_LOD0  (highest detail)
MeshName_LOD1
MeshName_LOD2
MeshName_LOD3  (lowest detail)

BlendMHS will automatically group these into a USD variant set.

Auto-Generated LODs

Enable automatic LOD generation with configurable settings:

  • LOD Levels: Number of LOD levels to generate (1-4)
  • Reduction Ratio: Polygon reduction per level (e.g., 0.5 = 50%)
  • Distance Ratios: Screen-space distances for LOD switching

Output

Meshes are exported to the meshes/ folder:

{project_path}/meshes/
