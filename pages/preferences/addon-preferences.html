Global addon settings, default paths, and configuration options that apply across all projects.

Overview

The Addon Preferences contain global settings for BlendMHS that persist across all Blender sessions and projects. Configure default paths, export behaviors, and debug options here.

Accessing Preferences

Access BlendMHS preferences through Blender's addon settings:

Edit ‚Üí Preferences ‚Üí Add-ons ‚Üí BlendMHS ‚Üí Preferences

Or use the quick access button in the BlendMHS panel header.

Settings

Default Project Path

Set a default directory for new BlendMHS projects. When creating a new project, this path will be pre-filled.

  • Path: Default directory for project exports
  • Tip: Use a shared network location for team projects

Export Defaults

Configure default export settings for new projects:

Setting Description
Apply Modifiers Default state for modifier application on export
Export UVs Include UV coordinates by default
Export Vertex Colors Include vertex color data by default
Auto-Export on Save Enable hot-reload workflow by default

LOD Defaults

Set default LOD generation parameters:

  • LOD Levels: Default number of LOD levels (1-4)
  • Reduction Ratio: Default polygon reduction per level
  • Auto-Generate LODs: Enable automatic LOD generation by default

Saving Preferences

Preferences are automatically saved when you close the preferences window. To ensure they persist:

  • Click "Save Preferences" in Blender's preferences window
  • Or enable "Auto-Save Preferences" in Blender settings
