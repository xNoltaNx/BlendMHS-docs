üîßMHS Toolkit

Meta Horizon Studio editor panel for automatic material processing and asset integration.

Overview

The MHS Toolkit is a companion panel that runs inside Meta Horizon Studio. It provides automatic processing of imported BlendMHS assets, ensuring materials are correctly configured and ready for use in your MHS project.

Panel Location

MHS Editor ‚Üí Window ‚Üí BlendMHS Toolkit

Features

Automatic Material Processing

Seamless Material Integration

When you import BlendMHS assets, the MHS Toolkit automatically:

  • Detects imported USD materials
  • Assigns the correct MHS renderer_module shaders
  • Links texture assets to shader inputs
  • Configures material properties to match Blender settings

Asset Browser Integration

The toolkit integrates with MHS's asset browser to provide:

  • Quick Import: Drag-and-drop BlendMHS USD files
  • Auto-Refresh: Detect changes when hot-reloading
  • Batch Processing: Process multiple assets at once

Hot-Reload Support

The MHS Toolkit fully supports MHS's hot-reload workflow:

  • File Watching: Monitors export directory for changes
  • Auto-Refresh: Updates assets when files change
  • Material Preservation: Maintains material assignments through reloads

Troubleshooting

Materials Not Processing

  • Verify the MHS Toolkit panel is open and active
  • Check that Auto-Process Materials is enabled
  • Ensure USD files are in a supported format

Textures Missing

  • Verify textures are exported to the textures/ folder
  • Check texture paths are relative in the USD file
  • Ensure texture formats are supported (PNG, JPG)
